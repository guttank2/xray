# XRay

Minimal Chrome extension that scrapes Twitter/X posts for content analysis.

## What it does

- Scans any Twitter/X profile
- Extracts: likes, retweets, replies, views, bookmarks, timestamps
- Exports to CSV

## Install

1. Download this repo (Code → Download ZIP)
2. Unzip
3. Create icons: convert `icons/icon.svg` to PNG (16, 48, 128px) at svgtopng.com
4. Chrome → `chrome://extensions` → Enable Developer Mode
5. Load unpacked → select folder

## Use

1. Go to any Twitter/X profile
2. Click XRay icon
3. Scan Posts
4. Export CSV
